## Design Notes

The original Stitch design system markdown describes a local-first UI direction called `The Digital Atelier`.

Key guidance:

- use tonal architecture instead of explicit borders
- deep forest green as the accent color
- off-white paper-like surfaces and soft charcoal text
- use value shifts and subtle texture to separate layout zones
- avoid SaaS-style gradients and generic shadows
- use a thin `signal bar` for active state instead of loud highlights
