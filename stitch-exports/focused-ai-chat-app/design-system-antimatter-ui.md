## Design Notes

The original Stitch design system markdown describes a calm, editorial, local-first UI language called `The Technical Curator`.

Key guidance:

- no hard section borders; use tonal separation, shadows, and spacing instead
- deep indigo as the anchor accent color
- wide, full-width AI response panels instead of chat bubbles
- large editorial typography with Manrope + Inter
- restrained gradients and glassmorphism only for selective overlays
- asymmetrical layout with breathing room and inspector-panel space
