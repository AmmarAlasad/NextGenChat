# Agent.md Template

This template represents the default `Agent.md` file created inside a new agent workspace.

It should contain:

- the agent's mission and operating constraints
- execution preferences and planning habits
- workspace-specific rules approved by admins
- references to related docs such as `identity.md`, `memory.md`, and `Heartbeat.md`

Future automation should copy and personalize this template per created agent rather than replacing the concept entirely.
